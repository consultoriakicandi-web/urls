TikTok OAuth - Teste

Autenticação para apps em modo teste

⚠️ App em Modo Teste: Como sua app está em teste, você precisa copiar manualmente o Access Token do TikTok Developer.
Obtenha em: https://developer.tiktok.com/apps/

📋 Copiar Access Token Manualmente

Step 1: Abra https://developer.tiktok.com/apps/
Step 2: Vá para "Tools" → "Access Token Generator"
Step 3: Clique em "Generate token"
Step 4: Copie o token gerado
O token é uma string longa começando com 'v1_...'

Preview do Token

📖 Como Funciona

🧪 Por que Modo Teste?

Quando sua app está em modo teste, o TikTok não permite o fluxo OAuth automático no navegador. É uma restrição de segurança do TikTok.

✅ Alternativas:

  • Opção 1 (Recomendada): Copiar token manualmente do Developer Dashboard (esta página)
  • Opção 2: Solicitar aprovação da app ao TikTok para sair do modo teste
  • Opção 3: Usar um backend (Node.js/Python) que fazer o OAuth server-side

🔑 O Token Salvo

Depois de salvo, o token é armazenado no seu navegador (Local Storage) e pode ser usado para publicar vídeos.

// Exemplo de uso com Python import json with open('tiktok_token.json') as f: data = json.load(f) access_token = data['access_token'] # Use o token para fazer requisições à API do TikTok
